# 🌐 คู่มือการติดตั้งและใช้งาน GradeInsights V5 (เวอร์ชันพร้อมใช้งาน)

โฟลเดอร์นี้ประกอบด้วยไฟล์สำเร็จรูปสำหรับใช้ติดตั้งระบบดูคะแนนสอบออนไลน์ เชื่อมต่อ Google Sheets โดยตรง

---

## 📁 ไฟล์ภายในโฟลเดอร์นี้
1. **`Code.gs`** — โค้ดสำหรับฝั่งหลังบ้าน (Backend) ใช้ติดตั้งบน Google Sheets Apps Script
2. **`index.html`** — หน้าเว็บแอปพลิเคชันสำหรับฝั่งหน้าบ้าน (Frontend) พร้อมใช้งาน

---

## 🛠️ ขั้นตอนการติดตั้งใช้งานใน 3 ขั้นตอน

### ขั้นตอนที่ 1: ติดตั้งโค้ดหลังบ้าน (Google Sheets)
1. เปิด **Google Sheets** ที่ต้องการใช้เก็บคะแนนนักเรียน
2. ไปที่เมนูด้านบน เลือก **ส่วนขยาย (Extensions) > Apps Script**
3. คัดลอกข้อความโค้ดทั้งหมดจากไฟล์ **`Code.gs`** ในโฟลเดอร์นี้ ไปวางทับในหน้าต่างโค้ดของ Apps Script (ลบโค้ดเก่าออกให้หมด)
4. กดปุ่ม **บันทึก (รูปแผ่นดิสก์)**

### ขั้นตอนที่ 2: เปิดใช้งานเว็บแอป (Deploy Web App)
1. ที่หน้าต่าง Apps Script คลิกปุ่ม **Deploy (ปรับใช้) > New deployment (การปรับใช้ใหม่)**
2. คลิกรูปฟันเฟือง เลือกประเภทเป็น **Web App (เว็บแอป)**
3. ตั้งค่ารายละเอียดดังนี้:
   * **Description:** พิมพ์ข้อความอะไรก็ได้ (เช่น เวอร์ชัน 1)
   * **Execute as (เรียกใช้ในฐานะ):** เลือกเป็น **Me (ตัวคุณครูเอง)**
   * **Who has access (ผู้มีสิทธิ์เข้าถึง):** เลือกเป็น **Anyone (ทุกคน)** *(สำคัญมาก)*
4. คลิกปุ่ม **Deploy (ปรับใช้)**
5. ทำการคัดลอกลิงก์ **Web App URL** ที่ระบบสร้างให้ (ลิงก์จะลงท้ายด้วย `/exec`)

### ขั้นตอนที่ 3: เปิดใช้งานหน้าเว็บดูคะแนน
* **วิธีใช้งานส่วนตัว/ทดสอบ:** ดับเบิ้ลคลิกเปิดไฟล์ **`index.html`** ในโฟลเดอร์นี้ผ่านเบราว์เซอร์ในเครื่องได้ทันที
* **วิธีอัปเดตหน้าเว็บใช้งาน:** 
  1. เมื่อเปิดหน้าเว็บขึ้นมาแล้ว ให้คลิกปุ่ม **"เข้าสู่โหมดคุณครู"** ที่มุมขวาบน
  2. กรอกรหัสผ่านเริ่มต้นคือ **`1234`**
  3. ไปที่แท็บ **"ตั้งค่าการเชื่อมต่อ (Settings)"** ด้านซ้ายมือ
  4. นำลิงก์ **Web App URL** ที่ได้จากขั้นตอนที่ 2 มาวางในช่องกรอกข้อมูล แล้วกด **"บันทึกการตั้งค่า"**
  5. ระบบหน้าเว็บจะเชื่อมต่อกับ Google Sheets ของคุณครูโดยตรงเรียบร้อยพร้อมใช้งานทันทีครับ!
